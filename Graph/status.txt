847. Shortest Path Visiting All Nodes                                B 
Here is the thought process behind this code:

This solution solves the "Shortest Path Visiting All Nodes" problem using **Breadth-First Search (BFS)** because it asks for 
the shortest path in an unweighted graph. Since we can revisit nodes, a simple `visited[node]` array isn't enough; the state 
must include **both the current node and the set of visited nodes so far**. We use a bitmask (integer) to represent the set of 
visited nodes (e.g., `101` means nodes 0 and 2 are visited). The BFS starts simultaneously from *all* nodes (adding them to the 
queue with their individual masks) to find the fastest route regardless of the starting point. We process the queue 
level-by-level, updating the mask as we move to neighbors (`currMask | (1 << neighbor)`), and return the path length as soon as 
any state reaches the `all` mask (where all bits are 1).

**Visualizing the State Space:**
Instead of just visiting nodes 0, 1, 2... imagine we are visiting states like `(Node 1, Visited {1})`, then moving to `(Node 2, Visited {1, 2})`.

133. Clone Graph                                              B
The thought process follows a standard DFS approach with memoization. First, you handle the edge case where the graph is empty 
(`NULL`). Then, you check your `visited` map to see if this node has already been cloned; if yes, you immediately return the 
stored copy to prevent infinite loops in cycles. If not, you create a brand new node with the same value and save it in the map 
(`old_ptr -> new_ptr`). Finally, you iterate through the original node's neighbors, recursively cloning each one and pushing 
the results into the new node's `neighbors` list, ensuring the entire structure is deeply copied before returning the new node.

Function Build(original_node):
    1. Base Case: If original_node is null, return null.
    
    2. Check Memory: If original_node is in Map, return Map[original_node].
    
    3. Create: Make new_node (copy of original_node's value).
    
    4. Save: Map[original_node] = new_node.
    
    5. Populate Children:
       For each child in original_node:
           new_node.child = Build(child)
           
    6. Return new_node.
